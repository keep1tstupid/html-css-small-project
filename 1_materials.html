Mozilla Development Network (MDN)

MDN Web Docs is an evolving learning platform for Web technologies and the software that powers the Web, including:

  • Web standards such as CSS, HTML, and JavaScript
  • Open Web app development
  • Firefox add-on development

MDN's mission is simple: to provide developers with the information they need to easily build projects on the open Web. If it's an open technology exposed to the Web, we want to document it.

Moreover, MDN Web Docs is available in 18 languages and allows users to add more translations.
